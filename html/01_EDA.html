1.2.2 Análisis de valores especiales¶

Estudiamos valores especiales del dataset.

En este dataset los valores especiales se clasifican en tres tipos:

  • Q: Este valor indica que se corresponde con una categora distinta a las dems.
  • N: Este valor indica que el criterio de valoracin no es aplicable. Este se corresponde generalmente, con tuplas que son valores dummies y por lo tanto no necesita esta informacin.
  • U: Este valor indica que se desconoce, un valor faltante.
  • X: Este valor indica que el valor est clasificado y no ha sido proporcionado.

Estos valores, si bien son universales al dataset, no estn presentes en todas las variables. Adems se presentan de distinta manera, pudiendo presentarse de los siguientes formatos: U, UU y UUUU.

Estos valores no representan la misma informacin y por lo tanto debern tratarse de forma distinta.

A continuacin, presentamos una tabla resumen de la existencia de estos valores en el dataset. Cada columna referencia el nmero de ocurrencias de cada valor (en cualquiera de sus formas) en la columna, el porcentaje sobre el total de valores de la columna, y tambin el total de los cuatro tipos.

In [ ]:
def special_values_summary(df = None, vals = []):
    # Conteo y porcentajes
    cols_svals_df = pd.DataFrame(data = df.columns, columns = ['column_name'])

    # Bucle de creacin de columnas de conteo
    for value in vals:
    
        totals = list()
        for col in cols_svals_df['column_name']:
            totals.append(df[df[col].isin(value)][col].count())
        
        cols_svals_df[value[0]] = totals
    
        percentajes = list()
        for col in cols_svals_df['column_name']:
            percentajes.append(np.round(cols_svals_df[cols_svals_df['column_name'] == col][value[0]] / 
                                        len(df.index) * 100, decimals = 3).values)
        
        cols_svals_df[value[0] + '%'] = percentajes
    

    cols_svals_df['total'] = cols_svals_df['Q'] + cols_svals_df['N'] + cols_svals_df['U'] + cols_svals_df['X']
    cols_svals_df['total%'] = np.round(cols_svals_df['total'] / len(df.index) * 100, decimals = 1)
    return cols_svals_df

Despus de este anlisis y el estudio del diccionario de datos, casi en su totalidad las columnas del dataset son categricas. A excepcin de las referentes a ao o meses. A continuacin, procederemos a analizar las variables categricas respecto a la variable objetivo seleccionada.

Al analizar la variable categrica, pudimos ver que existe un importante desiquilibrio, pues poseemos una gran cantidad de variables de accidentes no mortales, en comparacin con los accidentes con al menos una vctima mortal. Para poder comparar las variables categricas haremos una normalizacin por grupos, de tal forma que el conjunto de valores registrados de una categora se normalizar por el total de accidentes mortales o no mortales. Creamos para ello una funcin que reutilizaremos:

In [ ]:
def norm_category(df = None, obj_val = "", cat_val = ""):
    grouped = df.groupby([obj_val, cat_val]).count().iloc[:,1]
    grouped = grouped.reset_index()
    grouped.columns = [obj_val, cat_val, 'counted']
    grouped['group%'] = np.round(grouped['counted'] / 
                                 grouped.groupby(obj_val)['counted'].transform('sum') * 100, decimals = 3)
    return grouped

A raz de este anlisis hemos podido comprender mejor los datos que tenemos y sus principales caractersticas. Adems hemos detectado posibles problemas del mismo a los que tendremos que establecer algn tipo de solucin para poder utilizar estos datos en el modelado.

A continuacin enunciamos las conclusiones y principales ideas obtenidas:

  • Missing values: Existen un porcentaje considerable de valores especiales o missings. Ser necesario establecer un criterio de tratamientos de estos. Especialmente para aquellas variables altamente afectadas. Entre las posibles opciones, se valora aplicar un criterio de valor medio para los porcentajes elevados y una eliminacin en porcentajes bajos.
  • Variable objetivo: Hay un desbalanceo importante de la variable objetivo C_SEV. Existiendo un 98% de valores 2 y un 2% de valores 1 aproximadamente. Es por ello que ser necesario realizar un reequilibrado del dataset previo a cualquie transformacin o divisin.
  • Variables numricas: Entre las variables pramente numricas (no categricas) existe una baja correlacin. A excepcin de las variables C_YEAR y V_YEAR sobre las que existe una importante correlacin entre s, ser necesario eliminar una de ellas. Se discutir ms detallamente en el preprocesado.
  • Valores anmalos: En el anlisis de las variables numricas hemos detecado un valor extrao para la naturaleza del dataset. Este recoge informacin hasta el ao 2014, sin embargo, tenemos vehculos fabricados 2015. Estos valores debern de ser eliminados del dataset, pues no es posible que un vehiculo del ao 2015 se vea implicado en un accidente del 2014.
  • Variables categrias:
    • Hemos observado la relacin de las variables categricas del modelo con la variable objetivo. Que nos ha permitido determinar variables significativas para el modelo. Adems hemos observado algn que otro desbalanceo en los valores de la muestra que podra llevar a un sesgo.     • Estas variables ya han sido transformadas en numricas, lo que reduce su tratamiento a la hora de realizar un encoding. Si bien la existencia de muchas categoras en algunos casos, no recomienda utilizar tcnicas como OneHot.
